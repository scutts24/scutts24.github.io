I have played Minecraft on-and-off for around 10 years now, so I was excited to be able to use it in class. I built a replica of Springtrap, specifically his pixelated minigame form, which is shown in one of the Five Nights At Freddy's (FNAF) games. I decided to build a FNAF character for this project because, like Minecraft, I have enjoyed the FNAF franchise for a long time.

After building this statue, I used an application called Mineways to save it as an stl file and put it into FlashPrint 5 to prepare it for 3D printing.
